## 📊 技术实现

### 组件层级

```
GameLayout (z-index: 0)
├── TopBar (z-index: 100)
├── MainMap (z-index: 0)
├── GameBottomBar (z-index: 100)
├── PlayerAvatar (z-index: 200)
└── Teleport → body
    └── Overlay (z-index: 1000)
```

### 浮层系统

**使用 Teleport**:
```vue
<Teleport to="body">
  <Transition name="overlay">
    <div v-if="showOverlay" class="overlay-backdrop">
      <div class="overlay-content">
        <!-- 浮层内容 -->
      </div>
    </div>
  </Transition>
</Teleport>
```

**优势**:
- 浮层渲染在 body 下（避免层级问题）
- 全局可用（不受布局限制）
- 动画平滑（Transition 封装）
