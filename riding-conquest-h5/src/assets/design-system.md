# 骑行征服 - 国风设计系统

## 设计理念
**「千里江山·骑行天下」** - 将中国传统山水画意境与现代骑行运动结合

---

## 色彩体系

### 主色调
```
底色：#0D1B2A（黛蓝）- 深邃夜空，象征骑行者的坚持
主色：#1B263B（藏青）- 沉稳大气，用于主要容器
强调：#E0B1CB（淡紫）- 优雅点缀
金色：#D4AF37（鎏金）- 尊贵感，用于边框、按钮、重要信息
```

### 辅助色
```
朱红：#C41E3A - 阵营色/攻击状态/重要提示
青绿：#2E8B57 - 友方/安全区域/完成状态
米白：#F5F5DC - 内容区背景/文字底色
墨黑：#2C2C2C - 主要文字
```

### 渐变方案
```css
/* 金色渐变（按钮/边框） */
background: linear-gradient(135deg, #D4AF37 0%, #B8941F 100%);

/* 深蓝渐变（背景） */
background: linear-gradient(180deg, #0D1B2A 0%, #1B263B 100%);

/* 祥云渐变（装饰） */
background: linear-gradient(90deg, rgba(224,177,203,0.3) 0%, rgba(212,175,55,0.3) 100%);
```

## UI 组件规范

### 卡片容器
```
- 圆角：12px
- 背景：rgba(27, 38, 59, 0.95) + 米白内容区
- 边框：2px solid #D4AF37（金色）
- 阴影：0 4px 16px rgba(0,0,0,0.4)
- 装饰：四角云纹/回纹图案
```

### 按钮
```
主按钮：
  - 背景：金色渐变
  - 文字：墨黑 #2C2C2C
  - 圆角：8px
  - 边框：1px solid #B8941F
  - 悬停：亮度 +10%

次按钮：
  - 背景：透明
  - 文字：金色 #D4AF37
  - 边框：1px solid #D4AF37
  - 圆角：8px
```

### 列表项
```
- 高度：56px
- 背景：rgba(245, 245, 220, 0.1)
- 分隔线：1px solid rgba(212, 175, 55, 0.3)
- 悬停：背景亮度 +15%
```

## 装饰元素

### 传统纹样
1. **云纹** - 用于背景、分隔线
2. **回纹** - 用于边框装饰
3. **海浪纹** - 用于底部装饰
4. **卷轴** - 用于公告/任务面板
5. **印章** - 用于状态标识

### 图标风格
- 线条图标，金色描边
- 填充图标，低饱和度色彩
- 避免使用 emoji，使用 SVG 图标

---

## 动效规范

### 过渡时间
```
微交互：150ms - 按钮悬停、开关
中等：300ms - 页面切换、卡片展开
大型：500ms - 进场动画、庆祝效果
```

### 缓动函数
```
默认：cubic-bezier(0.4, 0, 0.2, 1)
弹性：cubic-bezier(0.68, -0.55, 0.265, 1.55)
```
