## 🎯 技术实现

### 数据源

**阿里云 DataV GeoJSON**：
- 国家级：`https://geo.datav.aliyun.com/areas_v3/bound/100000_full.json`
- 省级：`https://geo.datav.aliyun.com/areas_v3/bound/{province_code}_full.json`
- 市级：`https://geo.datav.aliyun.com/areas_v3/bound/{city_code}_full.json`

### 核心特性

**1. 动态加载**：
```js
// 按需加载 GeoJSON，缓存已加载数据
const loadGeoJson = async (level, code) => {
  const cacheKey = `${level}_${code}`
  if (geoJsonCache[cacheKey]) return geoJsonCache[cacheKey]
  // 加载并缓存...
}
```

**2. 领土着色**：
```js
// 根据领土归属自动着色
const getTerritoryStyle = (feature, territoryData) => {
  if (territoryData.is_ally) {
    return { areaColor: 'rgba(34, 197, 94, 0.35)' } // 绿色
  } else if (territoryData.team_id) {
    return { areaColor: 'rgba(239, 68, 68, 0.3)' }  // 红色
  }
  return { areaColor: 'rgba(107, 114, 128, 0.15)' } // 灰色
}
```

**3. 三级下钻**：
```js
// 根据点击自动判断下钻级别
const handleProvinceClick = (adcode, name) => {
  // 国家级点击 → 下钻到省
  drillDown('province', adcode)
}

const handleCityClick = (adcode, name) => {
  // 省级点击 → 下钻到市/县区
  drillDown('city', adcode)
}
```

### 数据格式

**领土数据**：
```js
[
  {
    code: '110101',        // 行政区划代码（6 位）
    name: '东城区',         // 名称
    team_id: 1,            // 战队 ID（无主为 null）
    is_ally: true,         // 是否我方
    defense_value: 100     // 防御值
  },
  // ... 2800+ 县区
]
```

## ⚠️ 注意事项

- 使用阿里云 DataV 公开数据源
- 首次加载需要网络请求（约 1-3MB）
- 已实现缓存机制，二次访问无需重新加载
