## 🎯 核心功能

### 1. 地图可视化区（左侧 2/3）

**功能：**
- 显示中国地图网格背景
- 领土标记点（带脉冲动画）
- 点击标记点选择领土
- 网格线效果（科技感）

**标记点样式：**
- ✅ 绿色 - 我方领土
- ✅ 红色 - 敌方领土
- ✅ 灰色 - 无主之地
- ✅ 蓝色脉冲 - 选中状态

### 2. 领土列表（右侧上 1/2）

**功能：**
- 可滚动列表
- 显示所有领土
- 点击选择领土
- 颜色区分状态
- 显示城防值

**交互：**
- 点击列表项 → 选中领土
- 选中项高亮（蓝色边框）
- 同步更新详情面板

### 3. 详情面板（右侧下 1/2）

**功能：**
- 显示选中领土详情
- 归属状态
- 城防值
- 战略价值（星级）
- 操作按钮（进攻/驻防/征服）

**按钮逻辑：**
- 无主之地 → "征服"（红色）
- 我方领土 → "驻防"（绿色）
- 敌方领土 → "进攻"（红色）

## 📐 响应式断点

### 横屏模式（激活）

```css
@media (orientation: landscape) and (min-width: 769px) {
  .landscape-layout {
    display: grid;
    grid-template-columns: 2fr 1fr;
  }
}
```

**触发条件：**
- 横屏方向
- 宽度 > 769px（平板/桌面）

### 竖屏模式（默认）

```css
.portrait-layout {
  /* 保持原有单列布局 */
}
```

**触发条件：**
- 竖屏方向
- 或 宽度 ≤ 768px（手机）

## 📊 对比优化前

### 竖屏模式（优化前）

| 问题 | 描述 |
|------|------|
| ❌ 地图被压缩 | 预览区很小，看不清细节 |
| ❌ 需要滚动 | 地图 + 列表需要纵向滚动 |
| ❌ 弹窗遮挡 | 详情弹窗遮挡地图 |
| ❌ 信息分离 | 地图和列表不能同屏 |

### 横屏模式（优化后）

| 优势 | 描述 |
|------|------|
| ✅ 地图完整 | 2/3 屏幕显示地图 |
| ✅ 信息同屏 | 地图 + 列表 + 详情一屏 |
| ✅ 操作流畅 | 点击即选，无需弹窗 |
| ✅ 战略感强 | 三分屏布局像指挥系统 |

## 🎯 技术实现亮点

### 1. 条件渲染

```vue
<!-- 横屏布局 -->
<div class="landscape-layout">
  <!-- 仅在横屏显示 -->
</div>

<!-- 竖屏布局 -->
<div class="portrait-layout">
  <!-- 仅在竖屏显示 -->
</div>
```

**CSS 控制：**
```css
.landscape-layout {
  display: none;
}

@media (orientation: landscape) and (min-width: 769px) {
  .landscape-layout {
    display: grid;
  }
  .portrait-layout {
    display: none;
  }
}
```

### 2. 状态管理

```vue
const selectedTerritory = ref(null)

function selectTerritory(territory) {
  selectedTerritory.value = territory
}
```

**单一数据源：**
- 地图标记点、列表项、详情面板共享同一状态
- 自动同步更新

### 3. 动态按钮

```vue
<button :class="getButtonColor()">
  {{ getAttackButtonLabel() }}
</button>
```

**逻辑：**
```js
function getAttackButtonLabel() {
  if (!selectedTerritory.value) return '选择领土'
  if (selectedTerritory.value.is_ally) return '驻防'
  if (selectedTerritory.value.team_id) return '进攻'
  return '征服'
}
```
