# 🐛 ChinaMap 组件修复说明

## 问题

1. **下钻后省份不在屏幕中央** - 需要手动拖动才能找到
2. **无法进入县区级视图** - 点击城市没有反应

## 已修复内容

### 1. 省份中心点自动聚焦

在 `drillDownToProvince` 函数中，使用 `provinceCenters` 字典获取省份中心坐标，并设置 `zoom: 2.5` 让省份充满屏幕。

```javascript
// 获取省份中心点坐标
const center = provinceCenters[provinceCode] || [104.0, 36.0]

// 更新地图配置 - 自动聚焦到省份中心
chartInstance.setOption({
  geo: {
    map: 'province',
    center: center,  // ✅ 自动聚焦
    zoom: 2.5,       // ✅ 更大的缩放比例
    regions: getGeoRegions()
  }
}, { lazyUpdate: true })
```

### 2. 三级下钻支持（全国→省→县区）

添加了：
- `cityAdcodes` - 城市/县区 adcode 映射
- `loadCityMap()` - 加载城市地图
- `drillDownToCity()` - 下钻到城市/县区
- `backToPrevious()` - 返回上一级

### 3. Bug 修复

**原代码**:
```javascript
currentProvinceCode.value = code  // ❌ code 未定义
```

**修复后**:
```javascript
currentProvinceCode.value = provinceCode  // ✅ 使用正确的变量
```

1. **不是所有城市都支持县区级下钻**
   - 目前只添加了部分主要城市的 adcode
   - 点击不支持的城市会显示警告但仍然更新 UI

2. **地图数据源**
   - 使用阿里云 DataV 的 GeoJSON 数据
   - 如果加载失败，检查网络连接

3. **缓存机制**
   - 已加载的地图会缓存到 `mapCache`
   - 再次访问同一地区时直接从缓存加载
