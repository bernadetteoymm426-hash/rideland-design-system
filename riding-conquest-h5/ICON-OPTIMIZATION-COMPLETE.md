## 🎨 图标设计规范

### 技术实现

```vue
<!-- SVG 图标组件 -->
<svg viewBox="0 0 24 24" 
     fill="none" 
     :stroke="color" 
     :stroke-width="strokeWidth" 
     stroke-linecap="round" 
     stroke-linejoin="round">
  <!-- 图标路径 -->
</svg>
```

### 设计参数

| 属性 | 值 | 说明 |
|------|-----|------|
| **尺寸** | 24x24 | SVG viewBox |
| **描边** | 2px | 默认 strokeWidth |
| **填充** | none | 空心图标 |
| **端点** | round | 圆角端点 |
| **转角** | round | 圆角转角 |

### 颜色系统

| 状态 | 颜色 | 效果 |

| **默认** | `#64748B` | 灰蓝色 |

## 🎯 图标设计详情

### 1. 首页图标（HomeIcon）
```
造型：城堡/基地
寓意：大本营、指挥中心
特点：对称结构、稳重感
```

**SVG 路径：**
- 屋顶三角形
- 主体矩形
- 门洞拱形

### 2. 疆域图标（MapIcon）
```
造型：地球经纬线
寓意：全国地图、领土
特点：圆形 + 经纬线交叉
```

**SVG 路径：**
- 外圆
- 经线（垂直）
- 纬线（水平）
- 大陆轮廓

### 3. 骑行图标（RideIcon）
```
造型：时钟/码表
寓意：速度、时间、骑行数据
特点：圆形 + 指针 + 刻度
```

**SVG 路径：**
- 外圆
- 中心指针
- 12 个刻度点

### 4. 战队图标（TeamIcon）
```
造型：多人组合
寓意：团队、协作、联盟
特点：多人剪影组合
```

**SVG 路径：**
- 左侧单人（大）
- 右侧单人（小）
- 头部圆形
- 身体弧线

### 5. 排行图标（RankIcon）
```
造型：列表/排行榜
寓意：排名、数据、对比
特点：多行结构
```

**SVG 路径：**
- 三行横线（列表）
- 左侧圆点（序号）

## 📊 优化对比

### 优化前（旧图标）

| 问题 | 描述 |
|------|------|
| ❌ **风格不统一** | 金色中国风 vs 科技蓝主题 |

### 优化后（新图标）

| 优势 | 描述 |
|------|------|
| ✅ **风格统一** | 科技蓝线条风格 |
| ✅ **SVG 格式** | 矢量图、无限缩放 |
| ✅ **高质量** | 清晰锐利、无锯齿 |
| ✅ **可定制** | 颜色/粗细可动态调整 |

## 🎨 图标样式系统

### 基础样式

```css
.icon {
  width: 28px;
  height: 28px;
  stroke: #64748B;          /* 默认灰蓝 */
  stroke-width: 2;
  fill: none;                /* 空心 */
  opacity: 0.6;
  filter: drop-shadow(0 2px 4px rgba(0, 0, 0, 0.3));
}
```

### 激活状态

```css
.icon.active {
  stroke: #3B82F6;          /* 科技蓝 */
  stroke-width: 2.2;         /* 稍粗 */
  opacity: 1;
  filter: drop-shadow(0 0 12px rgba(59, 130, 246, 0.6)); /* 发光 */
}
```

### 悬停状态

```css
.icon:hover {
  stroke: #94A3B8;          /* 浅灰 */
  opacity: 0.8;
}
```

## 🔧 技术实现

### 组件结构

```
src/components/icons/
├── HomeIcon.vue      # 首页图标
├── MapIcon.vue       # 疆域图标
├── RideIcon.vue      # 骑行图标
├── TeamIcon.vue      # 战队图标
└── RankIcon.vue      # 排行图标
```

### 使用方式

```vue
<script setup>
import HomeIcon from '@/components/icons/HomeIcon.vue'
</script>

<template>
  <HomeIcon class="icon" />
</template>
```

### 动态组件

```vue
<template>
  <component :is="getIconComponent(iconName)" class="icon" />
</template>

<script setup>
function getIconComponent(iconName) {
  const iconMap = {
    home: HomeIcon,
    map: MapIcon,
    ride: RideIcon,
    team: TeamIcon,
    rank: RankIcon
  }
  return iconMap[iconName]
}
</script>
```
