## 🎨 设计规范

### 统一样式
所有新页面必须使用：
- **背景：** `bg-conquest-abstract.jpg`（或纯色 #F5E6D3）
- **卡片：** `.card-guofeng` 类
- **按钮：** `.btn-gold` / `.btn-black` / `.btn-outline`
- **数字：** `.data-num`（金色渐变）
- **分割线：** `.divider-gold`

### 颜色方案
```css
--color-bg: #F5E6D3;        /* 玉脂白 */
--color-gold: #D4AF37;      /* 鎏金色 */
--color-bronze: #C9A961;    /* 古铜金 */
--color-ink: #2D2D2D;       /* 墨黑色 */
--color-cream: #FFF8E7;     /* 米白色 */
```
