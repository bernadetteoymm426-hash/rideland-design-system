## 🎭 渐变方案

### 金色渐变（文字/装饰）
```css
background: linear-gradient(180deg, 
  #FFD700 0%, 
  #D4AF37 50%, 
  #C9A961 100%
);
```

### 卡片背景渐变
```css
background: linear-gradient(135deg, 
  rgba(30, 58, 95, 0.55) 0%, 
  rgba(36, 64, 106, 0.6) 100%
);
```

### 按钮渐变
```css
background: linear-gradient(135deg, 
  rgba(30, 58, 95, 0.5) 0%, 
  rgba(36, 64, 106, 0.55) 100%
);
```

### 底部导航渐变
```css
background: linear-gradient(180deg, 
  rgba(30, 58, 95, 0.65) 0%, 
  rgba(15, 32, 59, 0.65) 100%
);
```

## 🎯 使用原则

1. **抽象 > 具象** - 背景用抽象纹理，不用具体图案
2. **融合 > 堆砌** - 颜色渐变过渡自然
3. **低对比度** - 整体柔和，不刺眼
4. **无文字** - 背景图不含文字
5. **金色点缀** - 重点信息用金色强调
