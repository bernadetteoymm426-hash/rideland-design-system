## 1. 设计愿景 (Design Vision)

### 核心定位
**「战略指挥屏」** - 这不是一个普通的游戏界面，而是一个沉浸式的领土攻防指挥中心。

### 设计关键词
- **沉浸感 (Immersion)**: 卫星夜景图视觉，深色底图 + 半透明领土着色
- **游戏化 (Gamification)**: 脉冲光效、进度动画、领土扩张可视化
- **科技感 (Technology)**: 蓝色光效、数据可视化、HUD 界面元素
- **情感化 (Emotion)**: 家乡高亮、集体叙事、每骑必反馈

### 设计原则
1. **地图即灵魂 (Map as Soul)**: 地图占据 85% 屏幕，是所有交互的核心载体
2. **反馈优先 (Feedback First)**: 任何用户动作都必须有即时视觉反馈
3. **情感先于逻辑 (Emotion Before Logic)**: 先触动用户情感，再解释规则
4. **一目了然 (Glanceable)**: 核心信息扫一眼就懂，核心操作点一下就行
5. **智能默认 (Smart Defaults)**: 系统替用户做决定，用户只需确认

## 3. 字体系统 (Typography)

### 字体家族 (Font Family)

**优先使用系统字体栈，零额外加载**：

```css
font-family: -apple-system, BlinkMacSystemFont, "PingFang SC",
             "Helvetica Neue", "Microsoft YaHei", sans-serif;
```

**理由**：
- 零加载时间（首包<200KB 约束）
- 中文渲染最优（PingFang SC 在 iOS、Microsoft YaHei 在 Android）
- 用户已熟悉的阅读体验

### 字号规范 (Type Scale)

| 级别 | 字号 | 行高 | 字重 | 用途 | 示例 |
|------|------|------|------|------|------|
| **Display** | 32px | 40px | Bold (700) | 攻城进度数字、核心数据 | "58%" |
| **H1** | 20px | 28px | Bold (700) | 页面标题 | "全国战况" |
| **H2** | 16px | 24px | Semibold (600) | 区块标题、卡片标题 | "本周战报" |
| **Body** | 14px | 22px | Regular (400) | 正文、列表项 | "浦东新区 · 攻城中" |
| **Caption** | 12px | 18px | Regular (400) | 辅助信息、时间戳 | "3 小时前更新" |
| **Micro** | 10px | 14px | Medium (500) | 地图标注、角标 | 地图上的县区名 |

### 数据展示规则

- **百分比数字（进度）**: Display 级，`#F1F5F9` 主色 + `#F59E0B` 高亮增量
- **里程数字**: H2 级，`#3B82F6` 蓝色
- **倒计时数字**: H2 级，等宽数字（`font-variant-numeric: tabular-nums`）防止跳动

## 5. 组件系统 (Components)

### 5.1 功能按钮 (Function Buttons)

**标准尺寸**:
- 宽度：100px
- 高度：80px
- 图标：32×32px
- 文字：13px

**视觉样式**:
```css
background: rgba(26, 39, 64, 0.6);
border: 1px solid rgba(59, 130, 246, 0.3);
border-radius: 12px;
icon-color: #3B82F6;
```

**5 种交互状态**:
1. **默认**: 深蓝背景 + 蓝色边框
2. **悬停**: 上浮 2px + 光效增强
3. **点击**: 下沉反馈 + 粒子效果
4. **禁用**: 灰色 + 无光效
5. **激活**: 更强蓝色光效 + 脉冲

- 左侧：Logo「骑迹大陆」+ 可选 Slogan

## 6. 动效系统 (Animation System)

### 缓动函数 (Easing Functions)

```css
/* 标准缓动 */
--ease-standard: cubic-bezier(0.4, 0, 0.2, 1);

/* 弹性缓动 */
--ease-bounce: cubic-bezier(0.68, -0.55, 0.265, 1.55);

/* 快速缓动 */
--ease-fast: cubic-bezier(0.4, 0, 0.6, 1);
```

### 持续时间 (Duration)

| 类型 | 持续时间 | 用途 |
|------|---------|------|
| **微交互** | 150ms | 按钮点击、小图标动画 |
| **状态过渡** | 300ms | 状态切换、颜色变化 |
| **页面切换** | 500ms | 浮层展开/关闭 |
| **大型动画** | 800ms | 镜头推进、家乡高亮 |

### 关键动画

**脉冲动画**:
```css
@keyframes pulse {
  0%, 100% {
    opacity: 1;
    transform: scale(1);
  }
  50% {
    opacity: 0.8;
    transform: scale(1.05);
  }
}
```

**上浮动画**:
```css
@keyframes float-up {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```

**缩放进入**:
```css
@keyframes scale-in {
  from {
    opacity: 0;
    transform: scale(0.95);
  }
  to {
    opacity: 1;
    transform: scale(1);
  }
}
```

## 7. 设计模式 (Design Patterns)

### 7.1 反馈模式 (Feedback Patterns)

| 模式 | 描述 | 使用场景 |
|------|------|---------|
| **顶部横幅** | 页面顶部非模态通知，3 秒自动消失 | 骑行数据同步、操作确认 |
| **进度跳动** | 数字从旧值动画到新值 + 地图领土颜色渐变 | 攻城进度变化、个人贡献增加 |
| **地图脉冲** | 相关区域短暂发光 | 领土变化、贡献计入 |

### 7.2 导航模式 (Navigation Patterns)

| 模式 | 描述 | 使用场景 |
|------|------|---------|
| **底部功能栏** | 固定底部 4 个按钮（战队/作战/战报/排行）+ 头像 | 一级页面导航 |
| **浮层详情** | 从底部展开的浮层（800px 宽，80vh 高） | 战区详情、成员详情、作战计划 |
| **全屏展开** | 从浮层扩展到全屏 | 战报详情、攻防详情 |
| **地图缩放** | 双指捏合缩放 + 点击县区 | 全国→省→县区三级下钻 |

### 7.3 决策模式 (Decision Patterns)

| 模式 | 描述 | 使用场景 |
|------|------|---------|
| **推荐 + 确认** | 系统推荐 → 用户一键确认 | 会长作战计划、大本营设置 |
| **默认 + 兜底** | 不操作也有合理结果 | 会长未宣战→自动执行推荐 |
| **先行动后理解** | 先让用户做一件事，再解释规则 | 新用户首次贡献骑行 |

## 9. 设计原则验证 (Design Principles Validation)

### UX 优先级框架

| 优先级 | 原则 | 含义 | 验证问题 |
|--------|------|------|---------|
| **P0** | 反馈优先 | 任何动作都必须有即时视觉反馈 | 用户操作后 150ms 内有反馈吗？ |
| **P1** | 情感先于逻辑 | 先触动用户情感，再解释规则 | 首次打开 5 秒内看到家乡吗？ |
| **P2** | 一目了然 | 核心信息扫一眼就懂 | 3 秒内能找到当前阶段吗？ |
| **P3** | 智能默认 | 系统替用户做决定 | 会长需要手动选择目标吗？ |
| **P4** | 地图即灵魂 | 地图是核心视觉和交互载体 | 地图占据 85% 屏幕吗？ |

### 情感化设计铁律

1. **永远正面表达** - 任何数据都用正向框架
2. **每骑必响** - 任何骑行后都有专属反馈
3. **紧迫不焦虑** - 营造期待感，不制造 FOMO
4. **集体叙事优先** - 个人数据放在集体故事语境中
5. **失败也有收获** - 即使输也展示积极面

   - 中国地图（85% 屏幕，领土着色）
   - 底部功能栏（4 个按钮 + 头像）
