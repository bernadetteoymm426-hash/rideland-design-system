# 项目名称变更通知

**变更日期**: 2026-03-20  
**原名称**: 骑行征服 (Riding Conquest)  
**新名称**: 骑迹大陆 (RideLand)

---

## 🎯 变更原因

1. **品牌升级** - 原名"骑行征服"较为直白，新名"骑迹大陆"更有故事感和 IP 价值
2. **游戏化强化** - "大陆"概念更强调世界观和领土征服的游戏体验
3. **差异化** - "骑迹"谐音"奇迹"，寓意骑行创造奇迹，更具情感共鸣
4. **扩展性** - 新名称更适合后续 IP 开发（副本、活动、剧情等）
