## 🔧 自定义生成

### 配置说明
- **用户数量**: 1-100（默认 20）
- **战队数量**: 1-20（默认 5）
- **每人骑行**: 1-50（默认 10）

### 示例配置
```
用户：50
战队：10
每人骑行：20
结果：50 用户 + 10 战队 + 1000 骑行记录
```

## 📋 API 调用

### 命令行使用
```bash
# 一键生成
curl -X POST http://localhost:8080/api/v1/mock/quick-setup

# 自定义生成
curl -X POST http://localhost:8080/api/v1/mock/generate \
  -H "Content-Type: application/json" \
  -d '{"user_count":50,"team_count":10,"rides_per_user":20}'

# 生成排行榜
curl http://localhost:8080/api/v1/mock/leaderboard
```

## ❓ 常见问题

### Q: 页面空白？
A: 检查路由配置，确保访问 /#/admin/mock

### Q: 数据不显示？
A: 生成数据后需要刷新页面

### Q: 如何清除数据？
A: 刷新页面即可，数据不持久化

### Q: 会影响真实数据吗？
A: 不会，这是纯模拟数据
