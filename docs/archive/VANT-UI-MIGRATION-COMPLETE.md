## 💡 使用技巧

### 主题定制
修改 `src/styles/variables.css` 即可全局改色

### 按需加载
自动导入，无需手动 import

### 图标使用
```vue
<van-icon name="star-o" />
<van-icon name="https://img.yzcdn.cn/vant/user-active.png" />
```

### 样式覆盖
使用 `:deep()` 覆盖 Vant 默认样式
