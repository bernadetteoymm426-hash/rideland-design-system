**完成时间**: 2026-03-08

## ✅ 功能概述

模拟数据管理系统允许你：
- ✅ 快速生成测试用户
- ✅ 创建模拟战队
- ✅ 生成骑行记录
- ✅ 创建对战数据
- ✅ 生成排行榜
- ✅ 无需真实骑行即可测试完整流程

---

## 🎯 访问方式

### 前端管理页面
访问：http://localhost:5173/#/admin/mock

### 后端 API
- POST `/api/v1/mock/quick-setup` - 一键生成
- POST `/api/v1/mock/generate` - 自定义生成
- GET `/api/v1/mock/leaderboard` - 排行榜
- GET `/api/v1/mock/user?id=1` - 单个用户

---

## 🚀 快速使用

### 方法 1: 前端页面（推荐）
1. 访问 http://localhost:5173/#/admin/mock
2. 点击"⚡ 一键生成测试环境"
3. 查看生成的数据预览
4. 刷新其他页面查看效果

### 方法 2: API 调用
```bash
# 一键生成（20 用户 +5 战队 +200 骑行）
curl -X POST http://localhost:8080/api/v1/mock/quick-setup

# 自定义生成
curl -X POST http://localhost:8080/api/v1/mock/generate \
  -H "Content-Type: application/json" \
  -d '{"user_count":50,"team_count":10,"rides_per_user":20}'

# 生成排行榜
curl http://localhost:8080/api/v1/mock/leaderboard
```
