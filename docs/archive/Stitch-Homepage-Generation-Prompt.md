## 📋 使用步骤

1. **打开 Stitch**：https://stitch.withgoogle.com/projects/5180537362465701192

2. **确保选择了设计系统**：
   - 点击右上角的 Design Systems
   - 选择 "Rideland Tactical"

3. **复制上面的提示词**（从 "Create a landscape..." 开始到 "...national map with territory control" 结束）

4. **粘贴到 Stitch 底部输入框**（显示 "What would you like to change or create?" 的位置）

5. **按回车或点击发送按钮**

6. **选择模型**（如果需要）：
   - 推荐：`3.0 Flash`（快速生成）
   - 或：`3.0 Pro`（更高质量）

7. **等待 30-60 秒**，Stitch 会生成 3 个变体

8. **选择最符合预期的设计**，点击 "Use this design"
