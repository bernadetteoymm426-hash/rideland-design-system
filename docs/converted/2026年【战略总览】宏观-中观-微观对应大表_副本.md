# 2026年【战略总览】宏观-中观-微观对应大表_副本

【公司级必赢之战】顽鹿生态——
主将：马高峰
一、主题：
三年内，深度融合AI，成为欧美核心市场最懂骑行运动用户的骑行助手之一
二、概述：
基于用户数据（骑行能力、疲劳状态、兴趣偏好）和环境数据（路线、高程、POI点、天气），通过构建运动体能算法和路线规划算法，进行路线的深加工，结合AI能力，为用户提供骑行运动兴趣专属的骑行助手
三、背景、目的及战略意义：
背景
1. 国内骑行市场深陷行业周期低谷，无论是整车还是配件的市场都面临巨大挑战，新增用户不足，阶段性大幅度投入的回报有限 ；海外市场在过去几年的渠道建设及产品改善基础上，呈现出高速增长态势，且增速的预期还会至少持续三年
2. 海外核心欧美市场的规模、成熟度、用户付费意识和能力都是在国内市场之上，对业务整体增长的价值更高 ；国内市场内卷严重，存在大量恶性竞争，且在下行周期过程中会日趋严重。同时我们看到国内的竞品都已经先与我们展开了出海业务，并取得良好成果
3. AI技术已经开始成为改变价值交付方式的重要推动力
4. 国内顽鹿业务目前相对稳定，在保障持续、有节奏的产研及运营投入下，战略防御的系统风险可控
目的
通过打造软硬一体的差异化高价值功能，让用户有加入平台的理由。为核心硬件码表带来卖点加持的基础上，提升顽鹿平台在海外尤其是核心欧美市场的用户黏性和用户规模
战略意义
1. 把握公司海外业务整体高增速发展的势能，提高顽鹿的用户价值，扩大有效海外用户的规模，提升公司数据资产价值
2. 通过打造差异化的高价值能力，持续累积用户和环境数据，优化训练体能和路线规划算法，深化训练AI能力，构建顽鹿的全球核心竞争力，为实现良性的用户商业化铺路
3. 欧美市场为全球核心市场，经过更高维度的历练并取得有效成果后，针对国内市场具有更高势能的影响力及竞争力，会为在国内周期上行阶段业务增长构建坚实基础
四、范围界定：
三年内聚焦目标和资源，优先打磨工具属性；
在价值更大的路线库及社交部分，三年内与寡头深度合作，避免正面竞争，挖掘和发挥生态价值；但在过程中需要持续构建相关产品及运营能力
两年内专注服务迈金/极遨码表用户，从第三年开始，通过手机码表的部分功能开放，吸引Strava等大型社区用户，并通过运营工作促进码表转化

【公司级必赢之战】顽鹿生态——
主将：马高峰
一、主题：
三年内，深度融合AI，成为欧美核心市场最懂骑行运动用户的骑行助手之一
二、概述：
基于用户数据（骑行能力、疲劳状态、兴趣偏好）和环境数据（路线、高程、POI点、天气），通过构建运动体能算法和路线规划算法，进行路线的深加工，结合AI能力，为用户提供骑行运动兴趣专属的骑行助手
三、背景、目的及战略意义：
背景
1. 国内骑行市场深陷行业周期低谷，无论是整车还是配件的市场都面临巨大挑战，新增用户不足，阶段性大幅度投入的回报有限 ；海外市场在过去几年的渠道建设及产品改善基础上，呈现出高速增长态势，且增速的预期还会至少持续三年
2. 海外核心欧美市场的规模、成熟度、用户付费意识和能力都是在国内市场之上，对业务整体增长的价值更高 ；国内市场内卷严重，存在大量恶性竞争，且在下行周期过程中会日趋严重。同时我们看到国内的竞品都已经先与我们展开了出海业务，并取得良好成果
3. AI技术已经开始成为改变价值交付方式的重要推动力
4. 国内顽鹿业务目前相对稳定，在保障持续、有节奏的产研及运营投入下，战略防御的系统风险可控
目的
通过打造软硬一体的差异化高价值功能，让用户有加入平台的理由。为核心硬件码表带来卖点加持的基础上，提升顽鹿平台在海外尤其是核心欧美市场的用户黏性和用户规模
战略意义
1. 把握公司海外业务整体高增速发展的势能，提高顽鹿的用户价值，扩大有效海外用户的规模，提升公司数据资产价值
2. 通过打造差异化的高价值能力，持续累积用户和环境数据，优化训练体能和路线规划算法，深化训练AI能力，构建顽鹿的全球核心竞争力，为实现良性的用户商业化铺路
3. 欧美市场为全球核心市场，经过更高维度的历练并取得有效成果后，针对国内市场具有更高势能的影响力及竞争力，会为在国内周期上行阶段业务增长构建坚实基础
四、范围界定：
三年内聚焦目标和资源，优先打磨工具属性；
在价值更大的路线库及社交部分，三年内与寡头深度合作，避免正面竞争，挖掘和发挥生态价值；但在过程中需要持续构建相关产品及运营能力
两年内专注服务迈金/极遨码表用户，从第三年开始，通过手机码表的部分功能开放，吸引Strava等大型社区用户，并通过运营工作促进码表转化
