# 科学训练框架_副本

科学训练框架
1.定义
使用智能硬件设备，通过训练(和日常)监控收集用户训练(和日常)数据，以提高训练质量、预防伤病、提高运动表现。

2.指标框架
根据运动能力提高原理，即“训练-反应-适应”过程，将所有指标分为四类：运动表现评估、训练、比赛、疲劳和恢复，它们互有联系并形成闭环。运动表现评估是闭环的开始，有评估，其他指标才能发挥作用。

3.指标类需求
1.通过历史训练数据评估运动表现⭐⭐⭐⭐⭐
FTP、FTHR、最大心率的准确性，是所有基于心率与功率衍生指标准确性的基础，直接决定了数据指标体系的可靠性。当前用户运动表现能力的设置依赖手动输入，需要构建自动能力评估机制：通过算法和图表追踪用户运动表现的动态变化，让用户清晰认知自身能力水平，驱动用户定期更新设置参数，从而持续获得更精准的训练数据。最大摄氧量和退耦率是衡量有氧能力的指标，临界功率和无氧储备是衡量无氧能力的指标。此类指标可以帮助用户了解自身能力并提供训练方向，以实现“训练-表现评估”的正反馈闭环，提升用户对训练体系的依赖与粘性。
2.计算非功率训练记录的负荷并对PMC曲线产生影响⭐⭐⭐⭐⭐
当前PMC只能计算功率训练负荷，无法覆盖只有心率数据或无心率、功率数据的训练场景，例如自行车训练体系中的关键训练：力量训练，体能训练等。为填补这一空白，需建立多维度训练负荷评估模型，针对不同类型的训练进行负荷统计，实现全面量化训练压力并加入PMC曲线的计算。进而更准确地监控体能疲劳的累积与恢复过程，避免过度训练或训练不足的情况。
3.评估训练效果⭐⭐
评估每笔记录的训练效果，并基于历史训练记录生成个性化的每日训练建议。
4.评估训练状态 ⭐⭐⭐
短时间的状态波动很大，评估整体训练计划中的状态以及每次训练时的状态，帮助用户了解自身状态并及时进行调整。
5.评估恢复能力⭐⭐⭐⭐
训练本身是对身体的破坏，真正的变强过程是在恢复中，它和训练同等重要。评估恢复能力需要睡眠、全天HRV等数据。
