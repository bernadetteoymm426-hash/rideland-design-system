📐 设计规范

色彩使用原则
• 主背景：深夜蓝 #0F1525（暗色页面）
• 次级背景：卷轴米白 #F5EBE0（浅色卡片）
• 强调色：古铜金 #C9A962（按钮/高亮/边框）
• 文字：米白 #F0E6D2（主）/ 灰蓝 #8B9BB4（次）
圆角规范
• 卡片：8-12px
• 按钮：22-30px（全圆角）
• 头像框：50%（圆形）
• 装饰元素：4-6px
边框使用
• 主边框：#2A3550（暗色卡片）
• 强调边框：#C9A962（金色，用于重要元素）
• 次级边框：#4A5D7A（用于输入框/次级按钮）
• 描边宽度：1-2px（常规）/ 3px（强调）
字体层级
• 页面标题：20-24px Bold
• 卡片标题：16-18px Normal
• 正文：14px Normal
• 次要文字：12-13px Normal
• 标注：11px Normal
装饰元素
• 云纹：#C9A962 金色（顶部/底部装饰）
• 波浪：#1A3A5C 青蓝（分隔线/底部）
• 传统纹样：用于卡片边角/边框装饰
间距规范
• 页面边距：16-20px
• 卡片内边距：16px
• 元素间距：12-16px
• 模块间距：24-32px
