# 骑行征服 - UI 线框图索引

**生成时间**: 2026-03-08  
**设计工具**: Pencil (.pen 格式)

## 📐 设计系统规范

### 色彩系统

| 类型 | 颜色 | Hex | 用途 |
|------|------|-----|------|
| **主色** | 深空蓝 | `#1a1a2e` | 页面背景 |
| **次级色** | 深夜黑 | `#0f0f1a` | 卡片/导航背景 |
| **强调色** | 科技蓝 | `#00d9ff` | 主按钮/高亮/进度条 |
| **警示色** | 警示红 | `#ff6b6b` | 错误/战斗/警告 |
| **成功色** | 生态绿 | `#00ff00` | 在线状态/成功提示 |
| **文字主色** | 纯白 | `#ffffff` | 主要文字 |
| **文字次级** | 中灰 | `#666666` | 次要文字/占位符 |

### 字体系统

| 用途 | 大小 | 字重 | 颜色 |
|------|------|------|------|
| 页面标题 | 20px | Normal | #ffffff |
| 卡片标题 | 16px | Normal | #ffffff / #00d9ff |
| 正文 | 14px | Normal | #ffffff / #666666 |
| 次要文字 | 12px | Normal | #666666 |
| 大数字 | 18-32px | Normal | #00d9ff / #ffffff |
| 按钮文字 | 16-18px | Normal | #1a1a2e / #ffffff |

### 圆角规范

| 元素 | 圆角 |
|------|------|
| 卡片 | 8-12px |
| 按钮 | 22-30px（全圆角） |
| 列表项 | 8px |
| 浮层 | 12px |

### 间距规范

| 类型 | 间距 |
|------|------|
| 页面边距 | 16px |
| 卡片内边距 | 16px |
| 列表项间距 | 8px |
| 模块间距 | 16-24px |

## 📱 页面列表

### 1. MapView（地图视图）

**文件**: `map-view.pen`  
**核心功能**: 全国地图展示、领土归属、队友位置

**布局结构**:
```
┌─────────────────────────────────┐
│ 周循环状态条 (60px)              │
├─────────────────────────────────┤
│                                 │
│         地图展示区               │
│    （高德地图 + 领土覆盖层）      │
│                                 │
│   [定位按钮] [缩放控制]          │
│                                 │
├─────────────────────────────────┤
│ 骑行贡献浮层（可选显示）          │
├─────────────────────────────────┤
│ 底部 Tab 导航 (100px)            │
│ 🗺️地图 👥战队 🚴骑行 ⚔️战斗 📊战报 │
└─────────────────────────────────┘
```

**关键组件**:
- 周循环状态条（常驻顶部）
- 地图容器（高德 SDK 集成）
- 领土多边形覆盖层
- 定位按钮（右下角）
- 缩放控制（右下角）
- 骑行贡献浮层（浮动卡片）
- 底部 Tab 导航
